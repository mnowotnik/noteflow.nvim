# Introduction

To create new notes use command
  :NoteflowNew <title>
To edit tags
  :NoteflowEditTags (or ,ne mapping )
To browse notes
  :NoteflowFind (or ,nf mapping )
To insert wikilink
  :NoteflowInsertLink (or ,nl mapping)
